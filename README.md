## INSTALL

- Download with git
```
git clone https://gitlab.com/oda-alexandre/vimrc.git ~/vimrc
```

- Move of vimrc in the folder /etc/vim/
```
mv -f ~/vimrc/vimrc /etc/vim/
```

- Supp of the residuts of install
```
rm -rf ~/vimrc
```
